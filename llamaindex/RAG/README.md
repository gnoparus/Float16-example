# Our Example Based on the Float16.cloud Model 

[Float16.cloud](https://float16.cloud)

## Why? 

Every language has its own unique characteristics. That's why we need to customize pipeline for each language.

## How? 
We combine the Float16.cloud API, Huggingface Embedding and LlamaIndex to create a new RAG pipeline for non-English languages.

## Environment 
- [LlamaIndex](https://www.llamaindex.ai/) (Data Framework)
- [Weaviate](https://weaviate.io/) (Vector Database)
- [SeaLLM-7b](https://huggingface.co/SeaLLMs/SeaLLM-7B-Chat) (AI Model) serving by [Float16.cloud](https://float16.cloud)
- [intfloat/multilingual-e5-large](https://huggingface.co/intfloat/multilingual-e5-large) (Embedding Model)
